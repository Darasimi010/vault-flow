# VaultFlow Feature: Expenses

This directory contains all expense management code including:
- Expense list and detail components
- Expense form with validation
- Expense API calls and queries
- Expense types and schemas

## Structure

```
expenses/
├── components/       # Expense-specific components
├── hooks/           # Expense-specific hooks
├── api/             # API calls for expense data
├── schemas/         # Zod validation schemas
└── types.ts         # Expense-specific types
```
